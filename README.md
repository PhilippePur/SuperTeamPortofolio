# Proyek Kolaborasi Website - Siklus 1 & 2

## üöÄ Siklus 1

1. **Project Manager (PM)** membuat **repositori sentral** di GitHub dengan file:
   - `README.md`
   - `index.html` (berisi menu dengan link ke halaman-halaman lain)

2. **Tiap anggota** melakukan **clone** dari repositori sentral ke **branch** di repo lokal.  
   - Penamaan branch: **menggunakan nama masing-masing anggota**

3. **Tiap anggota** membuat **file HTML** (nama file sesuai dengan link di `index.html`) berisi:
   - Nama
   - Foto diri (boleh diganti dengan gambar lucu)
   - Hobi
   - Keahlian (bahasa, framework, dll)

   Setelah selesai, lakukan **push** branch ke GitHub.

4. **PM** melakukan **review** dari tiap branch, lalu melakukan **merge** ke branch **main (master)**.

---

## üîÅ Siklus 2

1. **Tiap anggota** melakukan **pull** dari repositori sentral ke branch di repo lokal  
   untuk mendapatkan file HTML terbaru.

2. **Tiap anggota** mengedit **file HTML masing-masing** dengan menambahkan:
   - Konten aktivitas terakhir
   - Foto aktivitas tersebut  
   Setelah itu, lakukan **push** ke repo sentral.

3. **PM** melakukan **review** dari tiap branch, lalu melakukan **merge** ke branch **main (master)**.

4. **Setiap anggota** melakukan **pull** (atau clone) dari repositori sentral,  
   lalu **menjalankan website lengkap** di komputer lokal masing-masing.
